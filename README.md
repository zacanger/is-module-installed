# is-module-installed

Check if a module is installed in your project.

--------

## Installation

`npm i is-module-installed`

## Usage

```javascript
const isInstalled = require('is-module-installed')

isInstalled('is-module-installed') // => true
isInstalled('some-weird-thing') // => false
```
